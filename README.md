# FishFourmNew
使用ABP实现简单论坛

### 技术特点
#### 1.基于DDD设计
#### 2.应用服务层使用AOP技术缓存DTO对象
#### 3.webApi使用控制台托管
#### 4.多套系统，单点登录
#### 5.OAuth2 实现身份验证，授权服务
#### 6.Knockout.Js作为前端框架
